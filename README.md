# 🎮 MARIO BROS: BTS EDITION 💜

Un emocionante juego tipo Mario Bros con temática BTS. ¡Rescata a todos los miembros de BTS y sé un verdadero ARMY!

## 🎯 Objetivo del Juego

Completa 3 niveles rescatando a los miembros de BTS:
1. **Nivel 1**: Rescata a **Jin** 
2. **Nivel 2**: Rescata a **Suga** 
3. **Nivel 3**: Rescata a **J-Hope**

## 🎮 Controles

| Tecla | Acción |
|-------|--------|
| ⬅️ Flecha Izquierda / **A** | Mover a la izquierda |
| ➡️ Flecha Derecha / **D** | Mover a la derecha |
| **ESPACIO** / **W** | Saltar |
| **ESC** | Salir del juego |

## 💜 Características del Juego

### ✨ Mejoras Implementadas:
- **3 Niveles tematizados** con progresión de dificultad
- **Sistema de puntuación** con coins normales (10 puntos) y especiales (50 puntos)
- **Múltiples tipos de plataformas**:
  - Plataformas normales
  - Plataformas móviles (se mueven suavemente)
  - Plataformas peligrosas (¡cuidado!)
- **Enemigos dinámicos** que patrullan y cambian de dirección
- **Sistema de vidas** - tienes 3 oportunidades
- **Gráficos mejorados** con animaciones suaves
- **Física realista** de gravedad y saltos
- **Cámara dinámica** que sigue al jugador
- **Pantalla de inicio** interactiva
- **Interfaz visual** con HUD mostrando puntos, vidas y nivel actual

### 🎨 Temática BTS:
- Personaje del jugador en color morado BTS
- Miembros de BTS como objetivos a rescatar
- Colores característicos de cada miembro
- Pantalla inicial con los 7 miembros de BTS

## 📊 Sistema de Puntuación

| Acción | Puntos |
|--------|--------|
| Recoger moneda normal 💛 | 10 |
| Recoger moneda especial 💗 | 50 |
| Completar un nivel 🎯 | 500 |

## 🎲 Progresión de Dificultad

### Nivel 1: Jin (Fácil)
- Pocas plataformas
- 2 enemigos
- Ambiente básico

### Nivel 2: Suga (Medio)
- Más plataformas y movimiento
- Plataformas peligrosas
- 3 enemigos
- Mayor distancia

### Nivel 3: J-Hope (Difícil)
- Muchas plataformas complejas
- Múltiples plataformas móviles
- 5 enemigos
- Mayor distancia y velocidad

## 🚀 Cómo Ejecutar

1. Asegúrate de tener **Python 3.7+** instalado
2. Instala pygame (si no lo está):
   ```
   pip install pygame
   ```
3. Ejecuta el juego:
   ```
   python juego.py
   ```

## 💡 Consejos para Jugar

- **Salta a la velocidad correcta**: No saltes aleatoriamente, planifica tus movimientos
- **Evita enemigos**: Si tocas un enemigo perderás una vida
- **Recolecta monedas**: Intenta recoger todas las monedas para obtener más puntos
- **Usa las plataformas móviles**: Pueden ayudarte a alcanzar alturas mayores
- **Mantén la calma**: En el nivel 3 necesitarás precisión

## 👥 Personajes

- **💜 El Jugador**: Un valiente ARMY listo para rescatar a BTS
- **Jin (Lvl 1)**: Rosa claro - El primogénito
- **Suga (Lvl 2)**: Gris - El productor
- **J-Hope (Lvl 3)**: Dorado - El bailarín
